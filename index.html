RC Control

Connect to "BasedLED". This should return a "BluetoothRemoteGATTCharacteristic" in the console in a few seconds. If it does not, then the connection failed. Try disconnecting and reconnecting. If that does't work, power cycle the Arduino.

The intake only refreshes when running and robot movement commands. So if you want to turn it on/off, you need to move the robot. Intake speed can be changed in bleMotor.js
